## How to use

- Clone this repository
- Install Java
  21 [link](https://www.azul.com/downloads/?version=java-21-lts&os=windows&architecture=x86-64-bit&package=jdk#zulu)
- Build app:
  ```
  ./mvnw package
  ```
- Run app:
  ```
  ./mvnw exec:java
  ```
